# fantasy-football-score-predictor
This repository contains Python files for a fantasy football score predictor and calculator for NFL quarterbacks, wide receivers, and running backs. The score predictor takes data from the 2024-25 season and uses it to predict fantasy scores for these players based on their matchups (opposing defense), historical averages, and potential injuries. The score calculator is a simple input-output machine that asks the user to put in a player's stats in an individual performance and outputs a score based on ESPN Standard rules. 
How to use: 
Download or clone repository and run driver.py. The qb, rb, and wr predictors are contained in the helper file as functions. The driver file contains inputs, but it draws from the helper file.
